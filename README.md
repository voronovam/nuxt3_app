# Vue 3 + Pinia App

Single-page app on Vue 3, Vite, Vue Router, and Pinia.

## Requirements

- Node.js `22.18.0+`

## Setup

```bash
npm install
```

## Environment

Create `.env` from `.env.example` and set your OMDb API key:

```bash
cp .env.example .env
```

## Development

```bash
npm run dev
```

App runs at [http://localhost:8080](http://localhost:8080).

## Build

```bash
npm run build
npm run preview
```
